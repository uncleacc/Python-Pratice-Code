## 多线程

> python内置了threading.Thread多线程库
>
> from threading import Thread

实现多线程的两种方法

1. 

   ```python
   from threading import Thread
   
   
   def func1():
       for i in range(10000):
           print("一", i)
   
   
   def func2():
       for i in range(10000):
           print("二", i)
   
   
   def main():
       t1 = Thread(target=func1())
       # t2 = Thread(target=func2())
       t1.start()
       func2()
       # t2.start()
   
   
   if __name__ == '__main__':
       main()
   ```

2. 

   ```python
   from threading import Thread
   
   
   class MyThread(Thread):
       def run(self):
           for i in range(1000):
               print("1")
   
   
   def func():
       for i in range(1000):
           print("2")
   
   
   def main():
       t = MyThread()
       t.start()
       func()
   
   
   
   if __name__ == '__main__':
       main()
   ```
